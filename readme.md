# GapSuite
GapSuite is a platform for gap filling, which consists of two components: Gap-Aid and Gap-graph. 
## Gap-Aid
Gap-Aid enables gap filling using raw sequencing reads.You can get it from [Gap-Aid](https://github.com/panlab-bioinfo/Gap-Aid)
## Gap-graph
Gap-graph performs gap filling by utilizing the unitig graph generated by assembler.You can get it from [Gap-graph](https://github.com/panlab-bioinfo/Gap-graph)
